# business-task-management-demo

A responsive app for managing business tasks with demo features.

## Overview

Crea un'applicazione demo moderna e responsive per la gestione delle attività aziendali.

L'app deve includere:

Dashboard con statistiche e grafici di esempio.
Gestione utenti con elenco, ricerca, aggiunta, modifica ed eliminazione.
Sistema di autenticazione simulato (login/logout).
Sezione attività/progetti con stato (Da fare, In corso, Completato).
Notifiche e messaggi di esempio.
Tema chiaro/scuro.
Design professionale con componenti moderni e intuitivi.
Dati fittizi precaricati per consentire il testing immediato.
Validazione dei campi nei form e gestione degli errori.
Layout responsive per desktop, tablet e smartphone.

Tecnologie suggerite:

Frontend: React + TypeScript.
UI: Material UI o Tailwind CSS.
Backend simulato con dati mock JSON.

Obiettivo: fornire una demo completa per testare funzionalità CRUD, navigazione, interfaccia utente e flussi applicativi senza dipendenze esterne.
